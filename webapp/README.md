# Telegram Mini App — Крестики‑нолики

- React + Vite.
- Интеграция с Telegram WebApp SDK (отправка результата в бота через `sendData`).
- Победа: показ 5‑значного промокода и отправка уведомления боту.
- Поражение: отправка уведомления боту.

## Локальный запуск

1. `npm install`
2. `npm run dev`

В браузере откройте `http://localhost:5173`. Вне Telegram объект `Telegram.WebApp` недоступен, для этого добавлен безопасный stub (сообщения в консоль). Для полноценного сценария откройте мини‑приложение из Telegram‑бота.

## Сборка

```
npm run build
npm run preview
```
